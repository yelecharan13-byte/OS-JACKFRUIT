## Project Overview

This project implements a simple container runtime using user-space and kernel components.

It demonstrates:
- CPU workload using cpu_hog  
- Memory workload using memory_hog  
- System behavior using logs  

---

## Execution

### CPU Workload
cpu_hog program runs and consumes CPU continuously.

### Memory Workload
memory_hog program allocates memory continuously.
